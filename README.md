# DailyChip Todo (Flutter)

Flutter implementation bootstrapped from the PRD.

## Implemented in this pass
- Local email/password auth with salted SHA-256 hash
- SQLite persistence for users and tasks
- Task create/list/complete/delete
- Daily cap enforcement (`20` tasks per day)
- Reminder scheduling hook with default 10-minute reminder
- Theme persistence (light/dark/system)
- Recurrence support (`none`, `daily`, `weekly`, `custom`) and automatic next-task creation on completion

## Run
1. Install Flutter SDK 3.x and ensure `flutter` is in `PATH`.
2. In this folder:
   - `flutter pub get`
   - `flutter run`

## Notes
- Platform folders (`android/`, `ios/`) are generated by Flutter CLI.
- Since Flutter CLI was unavailable in this environment, source code was created directly.
